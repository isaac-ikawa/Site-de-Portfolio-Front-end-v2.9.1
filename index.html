Meu Projeto

Descrição do Projeto

Blog focado na história e culturas das artes marciais, oferecendo um design atraente e funcional.

Este projeto foi desenvolvido com o intuito de resolver problemas específicos e proporcionar uma experiência agradável aos usuários. Utilizei tecnologias modernas como HTML5, CSS3, JavaScript e jQuery para construir uma interface intuitiva e responsiva, aplicando técnicas de design responsivo como Grid e Flexbox.

Problemas e Soluções

Problema: Falta de responsividade em dispositivos móveis.
Solução: Utilize Grid e Flexbox para criar um layout flexível que se adapta a diferentes tamanhos de tela.

Problema: Bugs relacionados ao design responsivo de imagens e textos.
Solução: Ajustei as propriedades CSS para garantir que os elementos redimensionem corretamente em diferentes dispositivos

Problema: Centralização da div na seção de direitos autorais, que apresentava inconsistências em diferentes navegadores.
Solução: Apliquei ajustes nas propriedades CSS para assegurar uma centralização uniforme.

Desafios

Durante o desenvolvimento do projeto, enfrentei obstáculos e superei diversos desafios:

- Resolução de bugs relacionados ao design responsivo de imagens e textos.

- Centralização da div na seção de direitos autorais.

- Criação de um blog focado na história e culturas das artes marciais, oferecendo um design atraente e funcional.
